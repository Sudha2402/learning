Javascript Variables

A variable is a container for a value, which can be number, string, boolean, array, object, null or undefined.

Click "Try it" to adopt the first H1 element in the iframe.

Javascript has three types of variables: global, local, and parameter.

Global variables are declared outside any function, and they can be used by everyone.

Local variables are declared within a function and can only be used within that function.

Javascript Operators

An operator is a symbol that tells the compiler to perform specific mathematical or logical manipulations. The assignment operator (=) gives a value to a variable .

The arithmetic operators are used with numeric values to either add, subtract, multiply or divide one number from another. The assignment operator (=) assigns a value to a variable.

The comparison operators allow you to compare two values and return true or false.

The logical operators allow you to combine expressions and return true or false.
